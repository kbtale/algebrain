# Algebrain 🧠

Mathematics educational tool.

## 🛠️ Stack

- **Electron**
- **Node.js**
- **SQLite**
- **HTML5** / **Vanilla CSS** / **JavaScript**

## 🛠️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Latest LTS recommended)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/algebrain.git
   cd algebrain
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

Launch the application:

```bash
npm test
```

## 📂 Project Structure

- `main.js`: Electron process & window management.
- `db-logic.js`: Database query logic.
- `sqlite-handling.js`: SQLite connection wrapper.
- `public/`: Frontend assets.
- `sql/`: Database schema.
